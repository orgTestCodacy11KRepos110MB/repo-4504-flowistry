Expand description

Data structure for sharing spans outside rustc.

Rustc uses byte indexes to describe ranges of source code, whereas most Javascript-based editors I’ve encountered (e.g. VSCode) use character-based (really grapheme-based) indexes. This data structure helps convert between the two representations.
